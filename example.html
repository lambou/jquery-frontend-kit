JQuery Frontend Kit

Set of tools for jQuery frontend development.

Requirements

  • jQuery v3.3.1: fast, small, and feature-rich JavaScript library.
  • cleave v1.6.0 : JavaScript library for formatting input text content when you are typing.
  • jQuery loading-overlay v2.1.7: A flexible loading overlay jQuery plugin.
  • jQuery confirm v3.3.4: alerts, confirms and dialogs in one.
  • jQuery UI v1.12.1: curated set of user interface interactions, effects, widgets, and themes built on top of the jQuery JavaScript Library.
  • Slimselect v1.27.0: Advanced select dropdown.
  • Fontawesome: set of icons for web apps.

Toggle display

Apply the class display-toggle to the element that represent the toggle and add the selector of the container to display or hide based on the toggle, as the data-container attribute value.

URL Query parameters injection from inputs

Apply the class inject-query-param to a text or checkbox input and the value of the input will be injected in the targeted url as a query parameter.

Available properties for the input

  • data-param: query parameter name
  • data-target: the link element selector
  • data-url-attr: the attribute of the url on the link element. The default value is href.
  • data-value-attr: custom input value attribute. Use it to initiate default input value. The default value is data-value
